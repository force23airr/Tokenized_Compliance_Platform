# RWA Tokenization Platform

Institutional-grade infrastructure for tokenizing real-world assets (Treasuries, private credit, real estate) with built-in compliance and custody automation.

## Repository Structure

- `/docs` - Technical and business documentation
- `/contracts` - Smart contracts (Solidity)
- `/api` - Backend API (Node.js)
- `/frontend` - Web dashboard (React)
- `/infrastructure` - DevOps and deployment configs
