# Subscription Agreement Templates

## Investor Types

- `individual_accredited.md` - Individual accredited investors
- `entity_investor.md` - Corporations, LLCs, partnerships
- `trust_investor.md` - Trusts and estates
- `fund_investor.md` - Funds and institutional LPs

## Workflow

1. Investor completes KYC
2. AI determines investor type
3. Appropriate template selected
4. Variables populated
5. DocuSign for e-signature
